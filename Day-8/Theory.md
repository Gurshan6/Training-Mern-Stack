# Day 08 - Conditional Statements and Loops

## Overview

The eighth day of the MERN Stack training focused on **Conditional Statements** and **Loops** in JavaScript. These concepts are essential for controlling the flow of a program and performing repetitive tasks efficiently. Conditional statements allow programs to make decisions based on specific conditions, while loops enable the repeated execution of code until a given condition is met.

The session strengthened problem-solving skills by introducing different decision-making structures and looping techniques used in JavaScript programming.

---

## Topics Covered

### 1. Conditional Statements

Learned how JavaScript uses conditional statements to execute different blocks of code depending on whether a condition evaluates to true or false.

**Conditional Statements Covered:**

* `if`
* `if...else`
* `else if`
* Nested `if`
* `switch` statement

These statements are used to control program flow and implement decision-making logic.

---

### 2. Loops

Studied different looping statements that allow a block of code to execute repeatedly until a specified condition is satisfied.

**Loops Covered:**

* `for` loop
* `while` loop
* `do...while` loop

Also explored loop control statements that modify loop execution.

**Loop Control Statements:**

* `break`
* `continue`

These statements help improve code efficiency by reducing repetition and providing better control over loop execution.
