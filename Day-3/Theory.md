# Day 03 - Introduction to CSS

## Overview

The third day of the MERN Stack training introduced **Cascading Style Sheets (CSS)**, the language used to style and enhance the appearance of web pages. While HTML provides the structure of a webpage, CSS is responsible for its visual presentation, including colors, fonts, layouts, spacing, and responsiveness. The session covered the basics of CSS, different ways to apply styles, selector types, text formatting, the CSS Box Model, and display properties.

This session helped build a strong foundation for designing attractive, organized, and responsive web interfaces.

---

## Topics Covered

### 1. Introduction to CSS

Learned the purpose of CSS and its importance in web development. CSS allows developers to separate the design of a webpage from its HTML structure, making websites easier to maintain, modify, and visually appealing.

---

### 2. Methods of Applying CSS

Studied the three standard ways of adding CSS to HTML documents.

**Methods Covered:**

* Inline CSS
* Internal CSS
* External CSS

Each approach is suitable for different scenarios depending on the size and requirements of a project.

---

### 3. CSS Selectors

Explored different types of selectors that are used to target HTML elements for styling.

**Selectors Covered:**

* Universal Selector (`*`)
* Element Selector
* Class Selector (`.class`)
* ID Selector (`#id`)
* Group Selector
* Descendant Selector
* Child Selector
* Adjacent Sibling Selector
* General Sibling Selector

These selectors allow developers to apply styles efficiently and precisely to specific elements.

---

### 4. Typography

Learned various CSS properties used to improve the appearance and readability of text on web pages.

**Properties Covered:**

* `color`
* `font-family`
* `font-size`
* `font-style`
* `font-weight`
* `text-align`
* `text-decoration`
* `text-transform`
* `letter-spacing`
* `word-spacing`
* `line-height`

These properties help create attractive and readable content while improving the overall user experience.

---

### 5. CSS Box Model

Studied the CSS Box Model, which explains how every HTML element is represented as a rectangular box on a webpage.

**Components of the Box Model:**

* Content
* Padding
* Border
* Margin

A clear understanding of the Box Model is essential for controlling element size, spacing, and page layout.

---

### 6. Display Property

Learned about the CSS `display` property, which determines how HTML elements are displayed and positioned on a webpage.

**Display Values Covered:**

* `block`
* `inline`
* `inline-block`
* `none`

Each display value changes the way elements occupy space and interact with surrounding content.
