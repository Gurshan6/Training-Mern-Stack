# Day 04 - CSS Flexbox and Media Queries

## Overview

The fourth day of the MERN Stack training focused on **CSS Flexbox** and **Media Queries**, which are key technologies for building responsive and adaptable web layouts. Flexbox makes it easier to organize, align, and distribute elements within a container, while Media Queries allow webpages to adjust their appearance according to different screen sizes and devices.

The session highlighted the principles of responsive web design and demonstrated how Flexbox and Media Queries can be combined to create flexible, user-friendly, and device-compatible web interfaces.

---

## Topics Covered

### 1. Introduction to Flexbox

Learned about the **CSS Flexible Box Layout (Flexbox)**, a one-dimensional layout system designed to arrange and align elements efficiently inside a container.

Flexbox simplifies webpage layout creation by providing flexible positioning, spacing, and alignment of child elements.

---

### 2. Flex Container Properties

Studied the properties applied to the **parent (flex container)** that control the behavior of all flex items.

**Properties Covered:**

* `display: flex`
* `flex-direction`
* `justify-content`
* `align-items`
* `flex-wrap`
* `align-content`
* `gap`

These properties determine the direction, alignment, spacing, and wrapping of elements within the container.

---

### 3. Flex Item Properties

Explored the properties that are applied to **individual flex items** to customize their behavior inside a flex container.

**Properties Covered:**

* `order`
* `flex-grow`
* `flex-shrink`
* `flex-basis`
* `align-self`

These properties provide greater flexibility by allowing each item to behave independently when necessary.

---

### 4. Responsive Web Design (RWD)

Introduced the concept of **Responsive Web Design**, which ensures that websites automatically adjust their layout and content for desktops, laptops, tablets, and mobile devices, providing an improved user experience across different screen sizes.

---

### 5. Media Queries

Learned how **Media Queries** are used to apply different CSS styles based on device characteristics such as screen width, height, and orientation.

**Syntax:**

```css
@media (max-width: 768px) {
    /* CSS Rules */
}
```

Media Queries help developers create layouts that respond dynamically to various screen resolutions.

---

### 6. Combining Flexbox with Media Queries

Practiced using **Flexbox** together with **Media Queries** to build responsive webpage layouts.

For example, a row-based layout on larger screens can automatically change into a column layout on smaller devices, ensuring better readability and usability.
