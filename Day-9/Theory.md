# Day-09 - Functions and Arrow Functions

## Overview

The ninth day of the MERN Stack training focused on **Functions** and **Arrow Functions** in JavaScript. Functions are one of the most important programming concepts because they allow code to be reused, organized, and maintained efficiently. The session covered different methods of creating functions, passing parameters, returning values, and introduced **Arrow Functions**, a simplified function syntax introduced in ES6.

This session helped build a strong understanding of writing modular and reusable JavaScript code.

---

## Topics Covered

### 1. Introduction to Functions

Learned about **functions**, which are reusable blocks of code created to perform specific tasks. Functions help reduce code duplication, improve readability, and make programs easier to manage and maintain.

---

### 2. Function Declaration

Studied how to create functions using the `function` keyword.

**Syntax:**

```javascript
function functionName() {
    // code
}
```

Function declarations are the standard way of defining reusable functions in JavaScript.

---

### 3. Function Parameters and Arguments

Explored how **parameters** are used to receive input values inside a function and how **arguments** are passed when calling a function.

This allows functions to perform operations on different sets of data without rewriting the same code.

---

### 4. Return Statement

Learned the purpose of the `return` statement, which sends a value back to the place where the function is called.

The `return` statement makes functions more useful by allowing their results to be reused in other parts of a program.

---

### 5. Function Expressions

Studied **function expressions**, where functions are assigned to variables and can be used like any other value.

**Example:**

```javascript
const greet = function() {
    console.log("Hello");
};
```

Function expressions provide flexibility and are commonly used in modern JavaScript programming.

---

### 6. Arrow Functions

Learned about **Arrow Functions**, introduced in ES6, which provide a shorter and cleaner syntax for writing functions.

**Syntax:**

```javascript
const greet = () => {
    console.log("Hello");
};
```

Also understood the basic differences between traditional functions and arrow functions, including their syntax and common use cases.
