# Day 06 - CSS Grid Layout

## Overview

The sixth day of the MERN Stack training focused on **CSS Grid Layout**, a modern two-dimensional layout system used to create structured and responsive web page designs. Unlike Flexbox, which is mainly designed for arranging elements in a single direction, CSS Grid allows developers to manage both rows and columns simultaneously. This makes it an excellent choice for designing complex and well-organized webpage layouts.

The session covered the fundamentals of CSS Grid, including grid containers, rows, columns, spacing, and item placement, followed by the development of a complete webpage layout using Grid.

---

## Topics Covered

### 1. Introduction to CSS Grid

Learned about the **CSS Grid Layout Module** and its importance in front-end web development. CSS Grid provides an efficient way to organize webpage content into a flexible system of rows and columns, making layouts easier to build and maintain.

---

### 2. Grid Container

Studied how an HTML element becomes a grid container by using the following property:

```css
display: grid;
```

The grid container acts as the parent element and controls the positioning and alignment of all grid items placed inside it.

---

### 3. Grid Rows and Columns

Explored the properties used to define the structure of a grid layout.

**Properties Covered:**

* `grid-template-columns`
* `grid-template-rows`

These properties determine the number, size, and arrangement of rows and columns within the grid.

---

### 4. Grid Spacing

Learned how to create space between grid items to improve layout readability and appearance.

**Properties Covered:**

* `gap`
* `row-gap`
* `column-gap`

These properties help maintain consistent spacing between rows and columns.

---

### 5. Grid Item Positioning

Studied how to place and control individual grid items within the layout.

**Properties Covered:**

* `grid-column`
* `grid-row`

These properties allow elements to occupy specific positions or span multiple rows and columns, making complex layouts easier to design.

---

### 6. Creating a Webpage Layout with CSS Grid

Applied CSS Grid concepts to build a structured webpage layout containing the following sections:

* Header
* Sidebar
* Main Content
* Multiple Content Sections
* Footer

This practical exercise demonstrated how CSS Grid efficiently organizes webpage components into a clean and responsive layout.
