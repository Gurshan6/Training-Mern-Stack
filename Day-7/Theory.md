# Day 07 - Introduction to JavaScript

## Overview

The seventh day of the MERN Stack training introduced **JavaScript**, the programming language that adds functionality and interactivity to web applications. While HTML defines the structure of a webpage and CSS manages its appearance, JavaScript enables dynamic behavior and user interaction. The session focused on the core concepts of JavaScript, including variables, data types, and operators, which are essential for writing programs and building interactive web applications.

---

## Topics Covered

### 1. Introduction to JavaScript

Learned about **JavaScript**, a high-level, interpreted programming language widely used for front-end web development. It works together with HTML and CSS to create dynamic, interactive, and responsive web pages. The session also covered the importance of JavaScript in modern web development and how it enhances the overall user experience.

---

### 2. Variables

Studied how variables are used to store and manage data within JavaScript programs.

**Types of Variables Covered:**

* `var`
* `let`
* `const`

Also learned the differences between these variable declarations based on their scope, redeclaration, reassignment, and recommended usage in modern JavaScript.

---

### 3. Data Types

Explored the various data types available in JavaScript and understood how they are used to represent different kinds of values.

**Primitive Data Types:**

* String
* Number
* Boolean
* Undefined
* Null
* BigInt
* Symbol

**Non-Primitive Data Types:**

* Object
* Array
* Function

These data types help developers store and manipulate different forms of data efficiently.

---

### 4. Operators

Learned about the different categories of operators used to perform calculations, comparisons, assignments, and logical operations in JavaScript.

**Operators Covered:**

#### Arithmetic Operators

* `+`
* `-`
* `*`
* `/`
* `%`
* `**`

#### Assignment Operators

* `=`
* `+=`
* `-=`
* `*=`
* `/=`

#### Comparison Operators

* `==`
* `===`
* `!=`
* `!==`
* `>`
* `<`
* `>=`
* `<=`

#### Logical Operators

* `&&`
* `||`
* `!`

#### Increment and Decrement Operators

* `++`
* `--`

These operators are fundamental for performing mathematical operations, comparing values, and implementing logical conditions in JavaScript programs.
