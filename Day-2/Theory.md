
# Day 02 - HTML Forms

## Overview

The second day of the MERN Stack training focused on understanding **HTML Forms**, which are an essential part of building interactive websites. Forms enable web applications to collect information from users and send it for processing. Throughout the session, we explored various form elements, input types, and attributes that help create structured, user-friendly, and responsive forms.

This session provided a strong understanding of how user input is collected, validated, and managed in web applications.

---

## Topics Covered

### 1. Introduction to HTML Forms

Learned the purpose of the `<form>` element and how it serves as a container for different input controls. It is responsible for organizing user input and submitting data to the server.

---

### 2. HTML Form Elements

Studied the most commonly used HTML form elements, including:

* `<form>`
* `<label>`
* `<input>`
* `<textarea>`
* `<select>`
* `<option>`
* `<button>`

Each element plays an important role in collecting different types of user information.

---

### 3. Input Types

Explored various HTML input types designed for specific data entry requirements.

**Examples:**

* `text`
* `password`
* `email`
* `number`
* `date`
* `radio`
* `checkbox`
* `file`
* `submit`
* `reset`

These input types help improve accuracy and make forms easier to use.

---

### 4. Form Attributes

Studied several important attributes that improve form functionality and validation.

**Attributes Covered:**

* `id`
* `name`
* `placeholder`
* `required`
* `readonly`
* `disabled`
* `maxlength`
* `min`
* `max`

These attributes help control user input, enhance accessibility, and provide a better user experience.

---

### 5. Dropdown Menus

Learned how to create dropdown lists using the `<select>` and `<option>` elements. Dropdowns allow users to choose a single option from a predefined list, making forms more organized and user-friendly.

---

### 6. Text Area

Practiced using the `<textarea>` element to accept multi-line input such as feedback, comments, descriptions, and suggestions.

---

### 7. Buttons

Implemented different button types used in HTML forms:

* Submit Button
* Reset Button
* Normal Button

Each button performs a different function, such as submitting data, clearing form fields, or triggering custom actions.
